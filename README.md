# 🐾 NutriCat

**NutriCat** — це інтелектуальний вебзастосунок для аналізу та підбору персоналізованого раціону котів на основі їхніх індивідуальних параметрів із використанням штучного інтелекту.

## 🚀 Основний функціонал
* **ШІ-асистент (OpenAI API):** Збір анамнезу тварини через природний діалог та автоматичний розрахунок потреби в макронутрієнтах.
* **Управління профілями:** Створення, збереження та редагування карток домашніх улюбленців.
* **Генерація раціону:** Індивідуальні рекомендації щодо харчування, розрахунок порцій на основі породи, ваги, віку, стерилізації та рівня активності.
* **Аналітика здоров'я:** Виявлення критичних станів (наприклад, ожиріння) та генерація попереджень щодо необхідності візиту до ветеринарної клініки.

## 🛠️ Технологічний стек
* **Frontend:** Angular, TypeScript
* **Backend:** Python, Django, Django REST Framework
* **База даних:** SQLite
* **ШІ Інтеграція:** OpenAI API 

## ⚙️ Встановлення та запуск (Локально)

### 1. Клонування репозиторію
```bash
git clone [https://github.com/Stefysichka/nutricat-project.git](https://github.com/Stefysichka/nutricat-project.git)
cd nutricat-project
```

### 2. Налаштування Backend (Django)
```bash
cd nutricat-backend
python -m venv venv
# Активація віртуального середовища для Windows:
venv\Scripts\activate
# Встановлення залежностей:
pip install -r requirements.txt
# Запуск міграцій та сервера:
python manage.py migrate
python manage.py runserver
```
*Примітка: Для коректної роботи ШІ необхідно створити файл .env у папці бекенду та додати змінну OPENAI_API_KEY=твій_ключ.*

### 3. Налаштування Frontend (Angular)
Відкрийте новий термінал та виконайте:
```bash
cd nutricat-frontend
# Встановлення Angular CLI (якщо не встановлено глобально):
npm install -g @angular/cli
# Встановлення залежностей проєкту:
npm install
# Запуск сервера розробки:
ng serve
```
Після успішної збірки застосунок буде доступний у браузері за адресою http://localhost:4200/
